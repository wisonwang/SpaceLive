
# 新榜数据

随着新媒体，如短视频/直播电商/垂直内容平台，以及其他巨量流量入口app/平台的发展，不管是对内容生产者，电商从业者等其他互联网从业者，都带来了更多的机遇和挑战。要想把握住机遇，离不开对各市场数据的获取和分析，但对于大部分从业者来说，几乎很难获取到各个平台具体运营数据。
所以就诞生了类似“新榜”这类的，专门做流量平台公司的内容指标排行产品的公司。
